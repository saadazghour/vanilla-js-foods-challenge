# Vanilla JS

**Entirely with vanilla JavaScript** as coding test practice.

## Features
- Renders food items (emoji + name) directly into the DOM
- Click any item → instantly removed (event delegation)
- High performance using `DocumentFragment`

No build tools, no frameworks. pure vanilla JS!

## Requirements Fulfilled
- ✅ `Foods` class that takes root element + `foodData`
- ✅ Renders all items with root as parent
- ✅ Clicking a food item removes it from the list

## Tech
- HTML5
- Vanilla JavaScript
- CSS3 (for presentation)
